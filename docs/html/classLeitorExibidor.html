Descrição detalhada

Classe singleton responsável por ler e carregar arquivos .class.

Implementa funcionalidades para:

  • Carregar arquivos .class verificando integridade
  • Decodificar a estrutura do arquivo .class
  • Gerenciar endianness
  • Preencher a estrutura ClassFile

ClassFile * LeitorExibidor::readClassFile ( FILE * file)

Carrega e decodifica um arquivo .class.

Realiza:

  1. Verificação do magic number (0xCAFEBABE)
  2. Checagem da versão do arquivo
  3. Leitura e decodificação de todas as seções
Parâmetros
filePonteiro para o arquivo .class aberto
Retorna
Ponteiro para ClassFile carregado (deve ser desalocado pelo caller)
