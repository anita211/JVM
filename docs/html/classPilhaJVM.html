Descrição detalhada

Classe para representar a pilha da JVM. Uma pilha é responsável por conter frames. O seu limite é dado por FRAME_MAX_SIZE.

Essa classe é um singleton, ou seja, somente existe no máximo 1 instância dela para cada instância da JVM.
