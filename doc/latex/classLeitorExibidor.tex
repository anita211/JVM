Classe singleton responsável por ler e carregar arquivos .class. 

Implementa funcionalidades para\+:
\begin{DoxyItemize}
\item Carregar arquivos .class verificando integridade
\item Decodificar a estrutura do arquivo .class
\item Gerenciar endianness
\item Preencher a estrutura \doxylink{classClassFile}{Class\+File}
