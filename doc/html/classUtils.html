◆ verificarVersao()

double Utils::verificarVersao ( const ClassFile * classFile)
static

Mapeia a versão do arquivo .class para versão do JDK.

Conversão baseada na especificação:

  • 52 = J2SE 8
  • 51 = J2SE 7
  • 50 = J2SE 6.0
  • 49 = J2SE 5.0
  • 48 = JDK 1.4
  • 47 = JDK 1.3
  • 46 = JDK 1.2
  • 45 = JDK 1.1
Parâmetros
classFilePonteiro para a estrutura ClassFile
Retorna
Versão equivalente do JDK (ex: 1.8 para major_version=52)

◆ verificarVersaoClass()

double Utils::verificarVersaoClass ( const ClassFile * classFile)
static

Alternativa para mapear versão do arquivo .class.

Nota
Parece ser duplicada de verificarVersao(), pode ser consolidada
Parâmetros
classFilePonteiro para a estrutura ClassFile
Retorna
Versão equivalente do JDK

Converte os dados de versão obtidos da struct classFile para a convenção encontrada em https://en.wikipedia.org/wiki/Java_class_file#General_layout

Não Tratados Java SE 14 = 58 Java SE 13 = 57 Java SE 12 = 56 Java SE 11 = 55 Java SE 10 = 54 Java SE 9 = 53

Tratados J2SE 8 = 52 J2SE 7 = 51 J2SE 6.0 = 50 J2SE 5.0 = 49 JDK 1.4 = 48 JDK 1.3 = 47 JDK 1.2 = 46 JDK 1.1 = 45

Retorna
O valor equivalente à versão lida em *classFile.
